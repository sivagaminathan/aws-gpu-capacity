READ ME

###
Pre-requisites: 
In powershell, use "aws login" to establish aws cloud console credentials on the local machine
Without AWS login credentials shared with the local machine, the code will not be able to connect to ec2  

How to run the code : 
python -m uvicorn GetInstances:app --reload

This starts the server 
Navigate to : "127.0.0.8000" and see available endpoints
Use endpoints as described
